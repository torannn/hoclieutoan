# BankOnTap Errors Folder

Thư mục này chứa các file lỗi dạng Markdown để biên tập trực tiếp trên `BankOnTap/error-editor.html`.

## Luồng làm việc

1. Chạy nút **"Chia 4 section + fill null answers"** trong app editor.
2. Hệ thống sẽ tạo/cập nhật các file:
   - `unresolved-answers.md`
   - `invalid-diagrams.md`
3. Mở từng file, sửa nội dung và bấm **Lưu file đang sửa**.

## Ghi chú

- Nếu `BankOnTap/exam.json` đang rỗng, bước prepare sẽ báo lỗi và chưa tạo được danh sách đầy đủ.
